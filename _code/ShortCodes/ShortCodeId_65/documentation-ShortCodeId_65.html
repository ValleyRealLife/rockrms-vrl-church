Input a combination of 'firstname', 'lastname', 'birthdate', 'emailaddress', and 'mobilephone', the output will be either a PersonAliasGuid or 'tim_blank'.

Example Lava:

{[ PAGFF firstname:'tim' lastname:'seo' birthdate:'1996-03-19' emailaddress:'tseo@vrl.church' mobilephone:'(509) 919-0802' ]}

When using Lava Variables, either of these two formats work:

{[ PAGFF firstname:var_FirstName ]}
{[ PAGFF firstname:'{{ var_FirstName }}' ]}
