This ShortCode can identify a GroupType by its GroupId or GroupGuid. Therefore, the first parameter specifies whether the identifier will be an Id or a Guid, and the second parameter passes the identifier, (and the third parameter specifies which Column to output).

The first parameter is 'idtype'. The options are 'GroupId' or 'GroupGuid'.

The second parameter is 'identifier'. Pass either the 'Id' or the 'Guid' value.

The thid parameter is 'output'. Specify which Column you want returned.

Example Lava:

{[ GroupTypeFromGroupId idtype:'GroupId' identifier:'1' output:'GroupTypeName' ]}
