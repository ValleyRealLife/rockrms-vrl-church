The first parameter is 'url'. Paste the href in there.

The second parameter is 'newtab'. Choose either Y or N.

The thid parameter is 'label'. This is what the button will say.

Example Lava:

{[ ButtonPrimary url:'/StudentsCampRegistration?RegInsId=597' newtab:'Y' label:'Register for MOVE 2024 - Student Registration' ]}

Example Output:

Register for MOVE 2024 - Student Registration

Note:

The above example is something you'd write in the Ocucrrence Note field of an Event Occurrence (directly underneath its Linkages).

Advanced Users:

This ShortCode also has the 'id' and 'class' parameters, these are optional parameters.

The 'class' parameter will be appended after btn, btn-primary, and margin-t-md

Example Lava:

{[ ButtonPrimary id:'1' class:'custom--class' url:'/StudentsCampRegistration?RegInsId=597' newtab:'Y' label:'Register for MOVE 2024 - Student Registration' ]}
