This ShortCode can identify a Location by its Id or Guid. Therefore, the first parameter specifies whether the identifier will be an Id or a Guid, and the second parameter passes the identifier, (and the third parameter specifies which Column to output).

The first parameter is 'idtype'. The options are 'Id' or 'Guid'.

The second parameter is 'identifier'. Pass either the 'Id' or the 'Guid' value.

The thid parameter is 'output'. Specify which Column you want returned.

Example Lava:

{[ LocationFromIdentifier idtype:'Id' identifier:'1' output:'Name' ]}
